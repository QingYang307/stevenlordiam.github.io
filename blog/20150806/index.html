A New iOS Side Menu Demo

Recently I was developing an iOS app for leetcode, I found when displaying the content menu, RESideMenu is good at showing the parallax effect for iOS 7/8 app. So I decided to deploy it on my app. But when I was adding more UIViewController to it, it had a bug. You can find the detailed issue I posted here.

After some source code reading, I found the solution: creating a subclass of UIViewController instead of RESideMenu as the README of RESideMenu said. Because the new UIViewController will behave as a separate View rather than the RESideMenu behavior. So you cannot connect them.

Beside fixing this bug, I also add more view to this project. You can see a demo below.

I have open sourced this project as a fork of RESideMenu and you can find the source code on SideMenu.

Please make pull request or submit an issue if you have any advice.
